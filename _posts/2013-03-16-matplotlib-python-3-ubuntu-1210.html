--- layout: post title: 安裝 matplotlib (Python 3) (Ubuntu 12.10) date: '2013-03-16T01:54:00.000+08:00' author: Chih-En Wu tags: - Linux - Ubuntu - Python 3 modified_time: '2013-03-16T01:54:12.583+08:00' blogger_id: tag:blogger.com,1999:blog-6518933078301809282.post-476255551137542601 blogger_orig_url: http://trunone.blogspot.com/2013/03/matplotlib-python-3-ubuntu-1210.html --- 由下列指令安裝：
$ sudo apt-get install python3-dev python3-pip python3-setuptools libfreetype6-dev libpng12-dev python3-pyqt4
$ sudo pip-3.2 install matplotlib

在要使用 matplotlib 的程式前加入：
import matplotlib
matplotlib.use('Qt4Agg')


如果不想每次寫程式都多那兩行
可以把這個設定檔
http://matplotlib.org/users/customizing.html#matplotlibrc-sample
放到這個路徑
~/.matplotlib/matplotlibrc

再將裡面的
backend      : GTKAgg
修改為
backend      : Qt4Agg
