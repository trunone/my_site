$ sudo add-apt-repository ppa:upubuntu-com/java
$ sudo apt-get update
$ sudo apt-get install oracle-java7-installer
如果已經安裝過了其它的JDK 可以用下面的指令來切換
$ sudo update-alternatives --config java
$ sudo update-alternatives --config javac

UPDATE:
1. 下載 .tar.gz 版本的 JDK
2. 解壓到一資料夾
$ sudo tar xf jdk-*-linux-*.tar.gz -C /usr/share/jdk
3. 安裝
$ sudo update-alternatives --install "/usr/bin/java" "java" "/usr/share/jdk/jdk{JDK_VERSION}/bin/java" 1
$ sudo update-alternatives --install "/usr/bin/javac" "javac" "/usr/share/jdk/jdk{JDK_VERSION}/bin/javac" 1
$ sudo update-alternatives --install "/usr/bin/javaws" "javaws" "/usr/share/jdk/jdk{JDK_VERSION}/bin/javaws" 1
