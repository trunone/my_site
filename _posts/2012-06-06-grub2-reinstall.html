--- layout: post title: 重新安裝Grub2 date: '2012-06-06T12:24:00.001+08:00' author: Chih-En Wu tags: - Linux - Ubuntu modified_time: '2012-06-06T15:49:28.281+08:00' blogger_id: tag:blogger.com,1999:blog-6518933078301809282.post-8723253536511599108 blogger_orig_url: http://trunone.blogspot.com/2012/06/grub2-reinstall.html --- 使用 Ubuntu Live CD 開機後 在 Terminal 輸入下列指令
$ sudo mount /dev/sdaX /mnt 
$ sudo mount --bind /proc /mnt/proc
$ sudo mount --bind /dev /mnt/dev
$ sudo mount --bind /sys /mnt/sys
$ sudo chroot /mnt
※ /dev/sdaX 為根目錄所在分割

再依下列指令重新安裝 Grub2
# apt-get remove --purge grub-pc grub-common
# apt-get install grub-pc
# grub-mkconfig
# update-grub
# grub-install /dev/sda
