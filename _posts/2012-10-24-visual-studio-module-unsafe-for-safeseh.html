--- layout: post title: Visual Studio 出現 "module unsafe for SAFESEH image" 錯誤 date: '2012-10-24T02:02:00.001+08:00' author: Chih-En Wu tags: - Visual Studio - MASM modified_time: '2012-10-24T02:02:18.818+08:00' blogger_id: tag:blogger.com,1999:blog-6518933078301809282.post-8637590277814003848 blogger_orig_url: http://trunone.blogspot.com/2012/10/visual-studio-module-unsafe-for-safeseh.html --- 當 Visual Stuido 在 Build 時 如出現這個錯誤

通常是因為要 Link 的 lib 是用比較舊版的 Visual Studio 所編譯的

如果想強制關閉 Visual Studio 的檢查 可以依下列路徑

Project → Project Properties → Linker → Advanced → Image Has Safe Exception Handlers → No
